색상정보 호출

색상정보 호출방법

[ backgroundColor 방법 ]: div와 같은 곳에 배경색으로 보여주는 방법
[ text 방법 ]: 문자열로 hex색상 번호를 보여주는 방법

설정 함수

rk_colorVal=[{"id":"태그 아이디","type":"출력방법"}];

사용가능한 태그

div, input type=text, p 등등 모든 태그에 아이디값 설정해서 사용가능

type 종류

[ hex ] : hex색상 번호를 문자열로 출력(ex)#ff0000)
[ bk ] : css background-color 배경색으로 지정

소스코드 예제

<body> <div id="get_palette"></div> //색상정보 표시할 div <div id="color_div" style="width:70px; height:30px;border:1px solid #000;"> </div> <script src="rk_colorPalette_v1.0.js"></script> <script> //설정정보 입력(두가지설정을 모두 입력해도되고 //hex 혹은 bk 하나만 설정해도된다) rk_colorVal=[ {"id":"color_div","type":"hex"}, {"id":"color_div","type":"bk"}, ]; </script> </body>
